# k8s-deployment-automation

## 📌 Descrição
Projeto para provisionamento de infraestrutura na AWS utilizando Terraform e deploy de aplicação Nginx em Kubernetes com Helm e CI/CD via GitHub Actions.

---

## 🚀 Infraestrutura (Terraform)

### Pré-requisitos
- AWS CLI configurado
- Terraform instalado

### Inicializar e aplicar

```bash
cd terraform
terraform init
terraform apply
